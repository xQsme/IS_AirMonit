Particles   
HTTPVerb  URI With Example Linked  INFO 
GET  localhost/api/particles Get All Particles 
GET  localhost:80/api/particles/{particle}/day/{day} Get Particles on a certain day format "DD-MM-YYYY" 
GET  localhost:80/api/particles/{particle}/days/{days} Get Particles between 2 days provided 
GET  localhost:80/api/particles/{particle}/city/{city}/day/{day}  Get Particles in city, on day provided 
GET  localhost:80/api/particles/{particle}/summarize/day/{day}/hour  Get Summary of certain Particle provided, on the day given  
GET  localhost:80/api/particles/{particle}/summarize/city/{city}/day/{day}/hour  Get Summary of certain Particle provided, on the day and city given 
GET  localhost:80/api/particles/{particle}/summarize/days/{days}/"  Get Summary of a Particle between 2 days provided 
GET  localhost:80/api/particles/{particle}/summarize/city/{city}/days/{days}  Get Summary of a Particle between 2 days provided in city given 
 
 
  Cities   
HTTPVerb  URI With Example Linked  INFO 
GET  localhost:80/api/cities  Get All Cities 
GET  localhost:80/api/cities/{city}/sensors  Get all Sensors from city provided 
GET  localhost:80/api/cities/{city}/sensors/last  Get the Last Sensor Inserted  
GET  localhost:80/api/cities/incidents  Get all Incidents 
GET  localhost:80/api/cities/{city}/incidents  Get all Incidents from a certain city 
GET  localhost:80/api/cities/events  Get all Events Possible 
POST  localhost:80/api/cities/{city}/sensors  Insert a Sensor with Information from Request's Body 
POST  localhost:80/api/cities/{city}/incidents  Insert a Incident with Information from Request's Body
